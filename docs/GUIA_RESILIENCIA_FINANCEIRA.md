# Guia de Resiliência Financeira e Jurídica - Cred30

Este manual serve como diretriz para a gestão segura do capital e operação do sistema Cred30, focando em blindagem jurídica e eficiência tributária (pagar imposto sobre o lucro, não sobre o faturamento total).

## 1. O Conceito de Intermediação de Negócios

Para o fisco e para a justiça, a Cred30 **NÃO é um banco**. A Cred30 é uma **Plataforma de Intermediação**.
- **O Capital Social (R$ 42,00 da cota):** Não é seu. É dinheiro de terceiros sob sua custódia. Você não paga imposto sobre ele.
- **A Taxa Administrativa (R$ 8,00 da cota):** Esta é a sua receita bruta. Aqui é onde incide o imposto.

## 2. Como usar o Relatório Fiscal

No Painel Admin, o "Relatório Fiscal" já separa esses valores para você:
- **Gross Revenue (Faturamento Bruto):** Soma de todas as taxas (Marketplace, Cotas, Juros de Empréstimos). Este é o valor que o seu contador deve usar para emitir Notas Fiscais.
- **Volume Transitory (Volume de Custódia):** Dinheiro que entrou mas pertence aos investidores/cotistas. Este valor deve ser declarado apenas como "Entas de Terceiros" no seu balanço, sem tributação.

## 3. Blindagem Jurídica na Prática

- **SCP (Sociedade em Conta de Participação):** Se o volume crescer, formalize os investidores como sócios participantes de uma SCP. Isso protege o seu patrimonio pessoal e o deles.
- **Garantia Pignoratícia:** O sistema já usa as cotas como garantia dos empréstimos. Isso é legalmente sólido pois caracteriza um "penhor de direitos".
- **Termos de Uso:** Mantenha os avisos de risco sempre visíveis. A transparência de que "pode haver perda" é a sua maior defesa contra acusações de fraude.

## 4. Onde o Dinheiro Fica? (Segregação)

**Regra de Ouro:** Tente manter uma conta bancária (ou um saldo no sistema) específico para o **Capital Social**. Nunca use o dinheiro dos R$ 42,00 para pagar despesas da empresa (luz, servidor, pro-labore). Use apenas os R$ 8,00 (Taxa) para isso.
